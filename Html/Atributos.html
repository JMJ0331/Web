Atributos Globales

Los atributos globales son aquellos que pueden ser utilizados en cualquier etiqueta HTML. Algunos ejemplos de atributos globales son:

  • id: Define un identificador único para un elemento.
  • class: Define una o más clases para un elemento, que pueden ser utilizadas para aplicar estilos CSS.
  • style: Permite aplicar estilos CSS directamente a un elemento.
  • title: Proporciona información adicional sobre un elemento, que se muestra como una tooltip al pasar el cursor sobre él.

Atributos form

Los atributos form son específicos de los elementos de formulario en HTML. Algunos ejemplos de atributos form son:

  • action: Especifica la URL a la que se enviarán los datos del formulario cuando se envíe.
  • method: Define el método HTTP que se utilizará para enviar los datos del formulario (GET o POST).
  • enctype: Especifica el tipo de codificación que se utilizará al enviar los datos del formulario.
  • novalidate: Indica que el formulario no debe ser validado antes de enviarse.

Atributos input

Los atributos input son específicos de los elementos de entrada en HTML. Algunos ejemplos de atributos input son:

  • type: Define el tipo de entrada (texto, contraseña, correo electrónico, etc.).
  • placeholder: Proporciona un texto de ejemplo que se muestra dentro del campo de entrada cuando está vacío.
  • required: Indica que el campo de entrada es obligatorio y debe ser completado antes de enviar el formulario.
  • maxlength: Especifica la longitud máxima de caracteres que se pueden ingresar en el campo de entrada.
