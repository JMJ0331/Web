Etiquetas de Documento

Las etiquetas de documento en HTML son fundamentales para estructurar una página web. Estas etiquetas definen la estructura básica del documento y contienen otros elementos que conforman el contenido de la página. Las principales etiquetas de documento incluyen:

  • <html>:</html> - Esta etiqueta envuelve todo el contenido de la página web y define el inicio y el final del documento HTML.
  • <head>:</head> - Contiene metadatos sobre el documento, como el título, enlaces a hojas de estilo, scripts y otros elementos que no se muestran directamente en la página.
  • <body>:</body> - Contiene todo el contenido visible de la página web, incluyendo texto, imágenes, enlaces y otros elementos multimedia.

Etiquetas de Meta Datos

Las etiquetas de metadatos en HTML proporcionan información adicional sobre el documento que no se muestra directamente en la página web. Estas etiquetas son esenciales para definir características del documento, como su codificación, estilo y enlaces a recursos externos. Las principales etiquetas de metadatos incluyen:

  • <meta> - Define metadatos como la codificación de caracteres, descripción del sitio y palabras clave para motores de búsqueda.
  • <style>:</style> - Permite incluir estilos CSS directamente dentro del documento HTML.
  • <title>:</title> - Define el título del documento que aparece en la pestaña del navegador.
  • <base> - Establece una URL base para todos los enlaces relativos en el documento.
  • <link> - Se utiliza para enlazar hojas de estilo externas u otros recursos al documento HTML.

Etiquetas de Agrupación

Las etiquetas de agrupación en HTML se utilizan para agrupar contenido relacionado y aplicar estilos de manera más efectiva. Estas etiquetas son útiles para organizar el contenido y mejorar la presentación visual de la página. Las principales etiquetas de agrupación incluyen:

  • <p>:</p> - Define un párrafo de texto.
  • <hr>:</hr> - Se utiliza para crear una línea horizontal que separa contenido.
  • <ul>:</ul> - Define una lista desordenada.
  • <div>:</div> - Define un contenedor genérico para agrupar elementos.
  • <figure>:</figure> - se utiliza para agrupar contenido multimedia independiente, como imágenes, diagramas, código o audio, que se puede mover del flujo principal del documento sin afectar su significado.
  • <figcaption>:</figcaption> - Proporciona una leyenda para el contenido de un elemento <figure>.

Etiquetas Textuales

Las etiquetas textuales en HTML se utilizan para dar formato y estructura al texto dentro de una página web. Estas etiquetas permiten crear enlaces, resaltar texto y organizar el contenido de manera efectiva. Las principales etiquetas textuales incluyen:

  • <a>:</a> - Define un hipervínculo que enlaza a otra página o recurso.
  • <span>:</span> - Se utiliza para aplicar estilos en línea a una parte específica del texto.
  • <sub>:</sub> - Define texto en subíndice.
  • <sup>:</sup> - Define texto en superíndice.
  • <cite>:</cite> - Se utiliza para citar el título de una obra, como un libro, artículo o sitio web.

Etiquetas Semánticas

Las etiquetas semánticas en HTML proporcionan significado al contenido de una página web, mejorando la accesibilidad y el SEO. Estas etiquetas describen claramente la función del contenido que contienen, facilitando la comprensión tanto para los desarrolladores como para los motores de búsqueda. Las principales etiquetas semánticas incluyen:

  • <nav>:</nav> - Define una sección de navegación con enlaces a otras partes del sitio.
  • <section>:</section> - Representa una sección genérica de contenido temático.
  • <article>:</article> - Define un contenido independiente y autocontenido, como un artículo de blog o una noticia.
  • <header>:</header> - Representa el encabezado de una sección o página, que puede incluir títulos, logotipos y navegación.
  • <main>:</main> - Define el contenido principal del documento, excluyendo elementos repetitivos como barras laterales y pies de página.
  • <footer>:</footer> - Representa el pie de página de una sección o página, que puede incluir información de contacto, derechos de autor y enlaces relacionados.
