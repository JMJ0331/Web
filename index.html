Que es HTML?

HTML son las siglas de HyperText Markup Language (Lenguaje de Marcado de Hipertexto), el lenguaje estándar utilizado para crear la estructura de las páginas web. Se compone de un sistema de etiquetas que organizan el contenido, como textos, imágenes y enlaces, para que los navegadores puedan interpretarlo y mostrarlo de manera organizada y funcional.


Historia de HTML

HTML, el lenguaje de marcado para crear páginas web, fue creado por Tim Berners-Lee en 1989 para compartir documentos científicos, aunque su primera versión oficial, HTML 1.0, se publicó en 1991. Ha evolucionado a través de varias versiones, como HTML 2.0 (1995), que estandarizó muchas características, HTML 4.01 (1999), que fue un estándar durante los 2000, y la más reciente y ampliamente utilizada, HTML5 (2014), que introdujo elementos multimedia y semánticos como video, audio.


Editores de HTML

Los editores de HTML son herramientas de software para crear y editar código HTML, disponibles en dos tipos principales: los editores de texto (como Visual Studio Code, Sublime Text y Notepad++) y los editores WYSIWYG (What You See Is What You Get), como Adobe Dreamweaver. Los editores de texto permiten editar el código fuente directamente con funciones como resaltado de sintaxis y autocompletado, mientras que los WYSIWYG ofrecen una vista gráfica de la página mientras se crea.


Tu primer documento HTML

Para crear tu primer documento HTML, abre un editor de texto como el Bloc de Notas o un editor de código como Visual Studio Code, escribe el código básico que incluye la declaración, la etiqueta, y dentro de esta, las etiquetas y con contenido como h1 y h2, y luego guarda el archivo con la extensión .html (por ejemplo, index.html). Visualice las imagenes para mejor referencia.


Etiquetas en HTML

Las etiquetas HTML son fragmentos de código que se utilizan para estructurar y dar formato al contenido de una página web, indicando al navegador cómo mostrar cada elemento. Generalmente se componen de una etiqueta de apertura (ej. p) y una de cierre (ej. /p), aunque algunas son autocontenidas. Cada etiqueta tiene un propósito específico, como definir un párrafo, un encabezado, una imagen o un enlace.


Atributos en HTML

Los atributos de HTML son palabras especiales dentro de las etiquetas de apertura que proporcionan información adicional sobre un elemento, modificando su comportamiento o apariencia. Siempre constan de un nombre y un valor, con el formato nombre="valor", y se colocan inmediatamente después de la etiqueta de apertura. Se usan para añadir funcionalidad, como id para identificar elementos, src para imágenes, o href para enlaces.


Reglas fundamentales HTML

Las reglas fundamentales de HTML incluyen la estructura básica de un documento (DOCTYPE, html, head, body), el uso correcto de etiquetas de apertura y cierre con anidación coherente, y la convención de usar minúsculas para etiquetas y atributos. Es crucial especificar el tipo de documento con !DOCTYPE html para evitar el modo de compatibilidad del navegador, usar un solo h1 para el título principal, y organizar el contenido con sangría para mejorar la legibilidad.


Etiquetas para hacer un documento en HTML

Para hacer un documento HTML, se usan etiquetas como !DOCTYPE html para definir el tipo de documento, html como la raíz, head para metadatos como el title, y body para el contenido visible. Dentro del body, se utilizan etiquetas para estructurar el contenido, como p para párrafos, h1 a h6 para títulos y a para enlaces.
